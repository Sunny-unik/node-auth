# node-auth

> This project is describing that how to implementing an authentication system using Node.js.

## Key Features

- Encrypted Passwords
- Email verification via OTP
- Secure authentication tokens
- Session management

## Tech Stack

- Express as server
- MongoDB for store database
- Nodemailer for sending mails
